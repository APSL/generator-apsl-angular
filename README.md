# generator-apsl-angular

Generador de aplicaciones AngularJS siguiendo el estándar APSL.

## Instalación

- Instalar herramientas:
```bash
$ npm install -g grunt-cli bower yo
```
- Clonar el proyecto.
- Entrar dentro la carpeta.
- Instalar dependencias:
```bash
$ npm i
```
- Ejecutar:
```bash
$ sudo npm link
```
